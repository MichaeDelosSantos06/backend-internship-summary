# Backend Developer Internship - DOST CALABARZON

## Overview

Completed a Backend Developer internship where I contributed to the development and maintenance of an internal microservices-based application. Collaborated with senior developers to implement backend features, fix bugs, and maintain RESTful APIs while following industry best practices.

## Technologies

- Node.js
- TypeScript
- Express.js
- PostgreSQL
- Kafka
- Git / Gitea
- Podman CLI

## Responsibilities

- Developed and maintained backend API endpoints.
- Fixed bugs and implemented new features based on project requirements.
- Collaborated with senior developers using Git and Gitea for version control.
- Participated in sprint planning, daily stand-up meetings, and code reviews.
- Tested and maintained backend services.

## Key Learnings

During my internship, I gained valuable hands-on experience in professional backend software development, including:

- Developing and maintaining RESTful APIs using Node.js, Express.js, and TypeScript.
- Writing SQL queries and working with PostgreSQL for data storage and retrieval.
- Understanding the fundamentals of microservices architecture and inter-service communication using Kafka.
- Using Git and Gitea for version control, collaboration, branching, and pull requests.
- Managing development environments and services using Podman CLI.
- Debugging backend applications and identifying the root causes of issues.
- Worked in an Agile team using the Kanban methodology, participating in daily stand-up meetings, sprint planning, and collaborative discussions.
- Improving code quality by following coding standards and incorporating feedback from senior developers.
- Strengthening problem-solving, communication, and teamwork skills while working on a real-world software project.
